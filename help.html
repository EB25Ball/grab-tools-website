Grab Tools - Help

# .level Files

As explained here, levels in GRAB are stored in .level files. The files themselves are stored in your headset in Android/data/com.slindev.grab_demo/levels/user or Android/data/com.slindev.grablevels/user/levels/user for the tester version, and are named by the Unix timestamp of when they were created (Seconds since 00:00:00 UTC on 1 January 1970). These files are most simply the info about the level and a list of levelNodes (objects), and it is by editing these files that most of these tools work. The files themselves are in Google Protobuf format meaning they can't be modified directly, but by converting them to readable JSON format we are able to do so easily.

# Transferring Files

To transfer files to or from your headset to your computer, you will need to use a file transfer app such as Sidequest (Advanced installer). Once you have Sidequest installed, plug in your headset to your computer and open the files tab (Folder icon). From there, navigate to your levels/user folder in the grab files and either upload or download .level files.

# My Tools

Level JSON Editor - Edit .level files in JSON format

This tool allows you to edit .level files in JSON format, which is much easier to read and edit than the Protobuf format. To upload a file click Click me and select a file! and locate a .level file. A textbox will appear with the JSON contents of the file. You can then save the file with Download and it will be converted back to Protobuf format (.level) and saved to your PC. You also have the option to copy the JSON directly with Copy JSON, but this will not work in game until reconverted back to protobuf (.level).

Level Compiler - Compile .level files into one

This tool allows you to compile multiple .level files into one larger file and groups each individual level for ease of use. To upload files click Click me and select multiple files! and locate multiple .level files you would like to compile. You can then save the file with Compile and it will be saved to your PC. You also have the option to copy the JSON directly with Copy JSON, but this will not work in game until reconverted back to protobuf (.level).

Pixel Art - Create pixel art in grab

This tool allows you to create pixel art in grab. You can choose the quality of the image and the basic level details. To create pixel art, click Click me and select an image! and locate an image file (preferrably a png). You can then save the file with Pixelate and it will be saved to your PC. You also have the option to copy the JSON directly with Copy JSON, but this will not work in game until reconverted back to protobuf (.level).
